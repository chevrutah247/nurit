# NURIT

Modern Next.js website starter for Russian Junior N'shei Chabad.

## Local development
1. Install dependencies with `npm install`
2. Start with `npm run dev`
3. Open `http://localhost:3000`

## Deployment
- Push code to GitHub
- Import the repository into Vercel
- Add the production domain in Vercel project settings
